<h1>🌦️ Weather App</h1>

<p>A simple and responsive Weather App that shows real-time weather data using an API. Users can search for any city and get details like temperature, humidity, and wind speed.</p>

<h2>🚀 Features</h2>
<ul>
  <li>🌍 Search weather by city name</li>
  <li>🌡️ Real-time temperature display</li>
  <li>💧 Humidity and wind speed info</li>
  <li>🌦️ Weather condition icons</li>
  <li>⚠️ Error handling for invalid city</li>
  <li>📱 Fully responsive design</li>
</ul>
